Preamble

Hey all. I was working on a project and put together a script that many people would find helpful, something that will get rid of the black bars without distorting the screen.

In this tutorial you will learn:

  • What the application surface is.

Start with stretching

  • application_surface
  • display_set_gui_size()

    • To start, we need to change how gamemaker draws the game to the window. When we draw sprites and text, we are actually drawing to a special surface called the application surface. The application surface is created when the game starts, won't be removed from memory while the game is running and by default draws to the window between the draw events and the gui events.

    surface_resize(application_surface, window_get_width(), window_get_height())
display_set_gui_size(window_get_width(), window_get_height())

    The first line stretches the application surface across the window to remove the black bars. The second line does the same, but for the gui surface. It will warp, but we can fix it!
